## flappy-bird-demo 0.1.0

A playable Flappy Bird clone built with pygame-ce and a Typer CLI.

```sh
uv add flappy-bird-demo
```

### Features

- Full game loop: gravity, flap physics, pipe generation, collision detection, and score tracking
- Procedurally drawn bird sprite with wing/beak detail
- Configurable game constants via frozen `GameConfig` dataclass
- CLI entry point (`flappy_bird_demo`) powered by Typer; runs with `python -m flappy_bird_demo`
- Clean module split: `Bird`, `Pipe`, `Game`, `Renderer`, `InputHandler`, `Score`
- Pytest suite with parametrized tests; coverage across Python 3.12/3.13/3.14
- CI via GitHub Actions: Ruff lint, `ty` type check, test matrix, coverage reporting
- Security scanning: CodeQL, Dependabot, zizmor workflow audit
- Trusted publishing to PyPI with OIDC and SLSA provenance attestation
