# Installation

## Stable release

To install flappy-bird-demo, run this command in your terminal:

```sh
uv add flappy-bird-demo
```

Or if you prefer to use `pip`:

```sh
pip install flappy-bird-demo
```

## From source

The source files for flappy-bird-demo can be downloaded from the [Github repo](https://github.com/audreyfeldroy/flappy-bird-demo).

You can either clone the public repository:

```sh
git clone https://github.com/audreyfeldroy/flappy-bird-demo
```

Or download the [tarball](https://github.com/audreyfeldroy/flappy-bird-demo/tarball/main):

```sh
curl -OJL https://github.com/audreyfeldroy/flappy-bird-demo/tarball/main
```

Once you have a copy of the source, you can install it with:

```sh
cd flappy-bird-demo
uv sync
```
